Что внутри:
- готовый минимальный каркас Godot 4 .NET проекта
- Main сам спавнит игрока
- база с безопасной зоной
- камера приближается на базе
- полноэкранное меню по Tab
- игра ставится на паузу при открытии меню

Что делать у себя:
1. распаковать архив в папку проекта
2. открыть именно Godot .NET
3. открыть project.godot
4. если Godot спросит про C# проект/сборку — согласиться
5. нажать Build
6. запустить сцену

Клавиши:
- WASD / стрелки — движение
- Tab — открыть/закрыть меню
- E — действие interact заведено в Input Map, но пока не используется

Если Godot будет ругаться на кэш или старые файлы .godot:
- удалите папку .godot в своем проекте
- откройте проект заново
- снова Build
