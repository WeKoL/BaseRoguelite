# Как проверять версию 3.0

1. Открой `project.godot` через Godot .NET.
2. Нажми Build.
3. При ошибке сборки сначала смотри первый красный C# error, а не весь список сразу.
4. Для логики отдельно можно запускать тестовый проект `Tests/BaseRoguelite.Tests` через `dotnet test`, если установлен .NET 8 SDK.

Основные новые классы лежат в `Core/*Logic` и `Shared/*`. Они специально почти не завязаны на Godot-сцены, чтобы их можно было изучать и тестировать отдельно.
