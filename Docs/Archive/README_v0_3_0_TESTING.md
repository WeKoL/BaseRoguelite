# Проверка версии 0.3.0

## Ручной чек-лист

1. Удали `.godot`, `bin`, `obj`, если они остались от старых билдов.
2. Открой `project.godot` через Godot .NET.
3. Нажми Build.
4. Запусти главную сцену.
5. Проверь HUD: HP, стамина, еда, вода, зона.
6. Подбери ресурсы через `E` и удержание `E` у ресурсных объектов.
7. Вернись на базу и открой меню `Tab`.
8. Проверь `База → Станки`: должны быть рецепты 0.3.0.
9. Проверь улучшения базы: радиомачта и стены.
10. Проверь журнал: должен быть текст версии 0.3.0.
11. Проверь бой ЛКМ с врагами в опасной зоне.

## Автотесты

```powershell
dotnet test Tests/BaseRoguelite.Tests/BaseRoguelite.Tests.csproj
```

`Tests/BaseRoguelite.Tests/Version030/Version030BranchTests.cs`
