BaseRoguelite v3.0 visible patch

Заменить файлы в проекте по тем же путям.
Этот патч делает изменения версии 3.0 видимыми в игре:
- HUD сверху слева: HP, стамина, текущая зона, подсказка взаимодействия.
- В мире создаются добываемые точки: каменная жила, куча досок, металлолом.
- В мире создаётся тестовая опасная зона и тестовый враг.
- ЛКМ рядом с врагом выполняет ближний удар.
- База -> Станки теперь имеет 2 рабочие кнопки крафта из хранилища.
- База -> Лечение теперь имеет рабочую кнопку лечения.
- Журнал показывает тестовые цели, а не заглушку.
- Включены предыдущие компиляционные фиксы GatherResult и ItemCatalog.
