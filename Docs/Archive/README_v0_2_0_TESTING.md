# Проверка версии 0.2.0

1. Открыть проект в Godot .NET.
2. Нажать Build.
3. Запустить Main.tscn.
4. Проверить HUD: HP, стамина, еда, вода.
5. Проверить Shift: игрок ускоряется, стамина падает.
6. Проверить H: если HP/еда/вода не полные, расходник используется из инвентаря.
7. Подойти к ресурсам и удерживать E: камень, доски, металл, еда, вода должны добавляться в инвентарь.
8. Вернуться в зелёную безопасную зону: ресурсы уходят в хранилище.
9. Открыть Tab → База → Станки: проверить крафт и улучшения.
10. Проверить красную зону и бой ЛКМ с врагом.

Тесты через терминал, если установлен .NET SDK:

```powershell
dotnet test
```
