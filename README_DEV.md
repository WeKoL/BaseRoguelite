# BaseRoguelite — dev notes

## Открытие
1. Открыть `project.godot` в Godot .NET 4.6.x.
2. Нажать `Build`.
3. Запустить `Main/Main.tscn`.

## Управление
`WASD` / стрелки — движение. `E` — подобрать или взаимодействовать. `Tab` — меню. `Esc` — назад/выход. ПКМ по предмету — действия.

## Правило структуры
`Core` — чистая логика, `Features` — сцены и UI, `Content` — данные, `Tests` — тесты.
