Установка:
1. Заменить файлы из архива поверх проекта.
2. Закрыть Godot.
3. Удалить папки bin и obj в корне проекта.
4. Снова открыть проект.
5. Пересобрать C#.

Что должно работать:
- Tab открывает и закрывает меню целиком.
- Esc в корневом меню возвращает в игру.
- Esc в подменю возвращает на уровень назад.
- Крестик в корневом меню закрывает меню.
- Крестик в подменю возвращает в корневое меню.
- Вкладка База показывает карточки действий.
- Хранилище открывается как отдельный внутренний экран.
