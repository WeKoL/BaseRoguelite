# BaseRoguelite — версия 3.0

Версия 3.0 собрана как самодостаточный крупный апдейт поверх доступной версии 1.14. Внутри оставлены все старые механики и добавлен новый слой систем, который закрывает следующие шаги по веткам развития.

## Что добавлено

1. Техническая ветка: `FeatureReadinessChecker`, отчёт готовности фичи, `GameplayBalanceValidator`, предупреждения по слишком маленькому инвентарю/хранилищу.

2. Инвентарь и экипировка: перемещение/объединение слотов через `InventoryMoveLogic`, снимки слотов с индексами, фильтрация непустых слотов, сортировочный snapshot, расчёт состояния экипировки через `EquipmentLoadoutScoreCalculator`.

3. Хранилище: лимит записей, `TryAddItem`, `CanFit`, поиск по item id, планировщик авторазгрузки по категориям, уплотнение стаков.

4. Крафт: требования к уровню станка, оценка числа возможных крафтов, очередь крафта, тик таймера крафта, отмена крафта с возвратом ингредиентов.

5. Игрок: выносливость, прямой урон, коллекция статус-эффектов, голод/жажда через `SurvivalNeedsState`.

6. Добыча: ресурсные узлы с требуемым инструментом, таблицы лута, эффективность инструмента, восстановление зарядов ресурсной точки.

7. Бой: состояние врага, ближняя атака, расход выносливости, простая логика ИИ врага, кулдаун атаки врага и touch-attack.

8. Мир: таймер экспедиции, риск/награда по уровню опасности зоны, подсказка направления на базу, извлечение добычи из инвентаря в хранилище.

9. База: эффекты уровней построек, энергия базы, резервирование энергии, защита базы и проверка пробития обороны.

10. Сохранения: расширенная модель сохранения, валидация save-файла, миграция до версии 3, генерация имён слотов сохранений.

11. UI: подсказка веса инвентаря, сравнение прочности, подтверждение опасных действий, строки tooltip для веса и прочности.

12. Feedback: очередь событий обратной связи, текст событий, camera shake state, выбор звукового события, план feedback при уроне.

13. Квесты: награды за квест, одноразовое получение награды, цепочки квестов, разблокировка рецептов через награду.

14. Тесты: добавлены новые тесты под все новые системы версии 3.0. Сейчас в тестовом проекте 117 `[Fact]`-тестов.

## Важно

Системы версии 3.0 в основном добавлены как чистое C#-ядро, чтобы не ломать текущие Godot-сцены. Следующий безопасный шаг — подключать эти классы к UI и сценам постепенно: сначала крафт-очередь в меню базы, затем здоровье/выносливость на HUD, затем врагов и сбор ресурсов в мире.

Я не запускал `dotnet test` и сборку Godot в этой среде, потому что здесь нет установленного .NET SDK/Godot. Проверена структура проекта, отсутствие дубликатов классов и базовый баланс фигурных скобок в `.cs` файлах.
